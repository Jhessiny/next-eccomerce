## Getting Started

First, install and run the development server:

```bash
yarn
and
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies

- Next.js
- Typescript
- Tailwind
- React query
- Zustand
- Next-Auth
- Axios
- Framer motion
